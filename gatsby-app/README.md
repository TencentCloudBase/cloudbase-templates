# Gatsby.js 示例

参考文档：https://docs.cloudbase.net/quick-start/web/gatsby.html

## 手工部署

在此目录下运行：

```
cloudbase framework deploy -e <your-env-id>
```

## 一键部署

点击下方按钮一键部署：
